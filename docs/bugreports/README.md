# Примеры баг-репортов
## Проект: pigu.lt
Интернет-маркетплейс бытовой техники, электроники и товаров для дома. Ранее я работала с сайтом как контент-менеджер: наполняла каталог магазина, загружала товарные карточки и контролировала корректность отображения контента, попутно находя баги в позоватеской части и админ панели.

[PIGU-S-001 - Search results: отсутствует пагинация](./PIGU-S-001.md)

## Проект: Telegram (Destop)
Мессенджер с облачной синхронизацией сообщений, черновиков и настроек между устройствами. При работе с настольной и мобильной версией я исследовала поведение синхронизации черновиков и обнаружила баг, при котором удалённый черновик самопроизвольно возвращается через некоторое время.

[TEGA-D-001 - Черновик сообщения появляется в чате Telegram после его удаления в десктопной и мобильных версиях клиента](./TEGA-D-001.md)

## Проект: etiketid.ee
Сайт интернет-магазина, специализирующегося на продаже этикеточных принтеров и аксессуаров к ним. Настраивала фильтры для магазина и столкнулась с блокирующим работу багом.
