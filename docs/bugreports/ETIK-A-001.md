# ETIKETID-A-001 Admin login: невозможен сброс пароля администратора

## Summary
Невозможно восстановить пароль администратора через форму **Forgot password**.  
При первой попытке система сообщает об ограничении частоты (360 минут), несмотря на то что запрос выполняется впервые и письма для восстановления не отправляются.

## Description

### Environment
- Web version

---

### Steps to Reproduce
1. Перейти на сайт https://etiketid.ee/.
2. Открыть форму входа администратора.
3. Нажать **Forgot password** / «Забыли пароль».
4. Дождаться загрузки формы восстановления пароля.
5. Ввести валидный email администратора (аккаунт существует).
6. Отправить запрос на восстановление пароля.
7. Обратить внимание на сообщение об ошибке.
8. Подождать 360 минут.
9. Повторить попытку восстановления пароля с тем же email.

---

### Expected Result
- При первой попытке восстановления пароля:
  - запрос должен быть принят системой;
  - на email администратора должно быть отправлено письмо со ссылкой для сброса пароля.
- Ограничение по времени (rate limit) должно применяться **только после успешной отправки письма**.
- Администратор должен иметь возможность восстановить доступ к системе.
